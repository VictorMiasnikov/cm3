Pragmas

A pragma is a compiler directive. It is an arbitrary sequence of characters bracketed by <* and *>.

  • INLINE Pragma
  • ASSERT Pragma
  • TRACE Pragma
  • FATAL Pragma
  • UNUSED Pragma
  • OBSOLETE Pragma
  • NOWARN Pragma
