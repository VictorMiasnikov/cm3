Expressions

Modula-3 supports a rich variety of strongly type-checked expressions.

  • Expression Building Blocks: Various identifiers and literals
  • Arithmetic Expressions: a + b, MAX(x,y), ABS(x), ...
  • Logical Expressions: a AND b, NOT c, ...
  • Relational Expressions: a = b, a <= c, c IN chart-set, ...
  • Constructors: Create array, and record initializers.
  • Precedence: Operator precedence chart
