Reference Types

    REF type

Type can be any type, even a reference type. This specifies a reference (pointer) to a value of type type.

Example

    TYPE
  IntPtrType = REF INTEGER;                 (* A pointer to an integer *)
  ArrPtrType = REF ARRAY [1..5] OF CHAR;    (* A pointer to an array *)

VAR
  p := NEW(IntPtrType);

BEGIN
  p^ := 10;
END;
