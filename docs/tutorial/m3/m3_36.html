WHILE Statement

    WHILE expr DO stmts END

expr must be a boolean-valued expression. stmts are executed until expr becomes FALSE. If expr is FALSE at the start stmts are not executed at all.

Example

    WHILE NOT IO.EOF() DO
  IO.Put (IO.GetLine());
END;
