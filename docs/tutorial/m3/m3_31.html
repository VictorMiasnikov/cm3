IF Statement

    IF expression THEN statements { ELSIF expression THEN statements ... } 
[ ELSE statements ] END

expression must be of type BOOLEAN.

Example

    IF Text.Equal(txt,"Strawberry") THEN
  IO.Put("Not bad")
ELSIF Text.Equal(txt,"Avacado") THEN
  IO.Put("Yuck!")
ELSIF Text.Equal(txt,"Chocolate") THEN
  IO.Put("Yum")
ELSE
  IO.Put("Nah.")
END
