Types

    TYPE identifier "=" type ";"

A type declaration allows you to give a name to a type. Type may be one of the following:

  • Ordinal: INTEGER, CARDINAL, BOOLEAN, CHAR, subrange, enumerated
  • Floating point: REAL, LONGREAL, EXTENDED
  • Array: a structure containing a number of elements of the same type
  • Record: a structure containing a fixed number of elements of possibly different types
  • Object: a record paired with a method suite
  • Set: a collection of ordinal values
  • Reference: a pointer to something
  • Procedure: a procedure signature

To see more information, follow one of:

  • Ordinal Types: INTEGER, CARDINAL, BOOLEAN, CHAR, ...
  • Array Types: ARRAY indextype OF slottype
  • Record Types: RECORD fieldlist END
  • Object Types: OBJECT-METHODS-OVERRIDES-END
  • Set Types: SET OF basetype
  • Reference Types: REF <referent type>
  • Procedure Types: PROCEDURE (parameter-list) { : return-type }
