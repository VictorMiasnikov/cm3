Language Elements

Modula-3 programs include declarations, statements, expressions, and pragmas.

  • Compilation Units: Modules and interfaces
  • Declarations: Types, constants, variables, procedures...
  • Statements: Selection, looping, error control
  • Expressions: Identifiers, arithmetic, logic, etc.
  • Pragmas: Compiler directives: <*INLINE*>, etc.
