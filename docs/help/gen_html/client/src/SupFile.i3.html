 This module provides a facility for parsing a supfile. 


INTERFACE SupFile;

IMPORT Pathname, SupFileRec, SupFileRecSeq;

EXCEPTION Error(TEXT);

PROCEDURE Parse(fileName: Pathname.T;
		override: SupFileRec.T := NIL;
		mask := SupFileRec.Options{}): SupFileRecSeq.T
  RAISES {Error};

 Parses a supfile, and returns its sequence of records.  If
   override is specified, its non-default client-controlled fields
   override anything specified in the supfile.  Only the options
   specified in mask are overridden. 

END SupFile.
