INTERFACE M3ID;

IMPORT M3Token;

PROCEDURE Classify (READONLY x: ARRAY OF CHAR): M3Token.T;

 If x is a Modula-3 keyword, return its token class,
   otherwise return M3Token.Ident.  It is a checked runtime
   error if NUMBER(x) is less than one. 

END M3ID.
