INTERFACE M3Keyword;

IMPORT M3Scanner;

PROCEDURE Classify (READONLY x: ARRAY OF CHAR): M3Scanner.TK;

 If x is a Modula-3 keyword, return its token class,
   otherwise return M3Scanner.TK_Ident.  It is a checked runtime
   error if NUMBER(x) is less than one. 

END M3Keyword.
