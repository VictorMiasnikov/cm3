 A Work.T is a piece of data representing a unit of work consumed by
   a Worker.T.  It has a separate interface so it can be used as a
   parameter to generic data structure modules.



INTERFACE Work;

IMPORT Thread;

CONST Brand = "Work";

TYPE
  T = OBJECT
    METHODS
      handle() RAISES {Thread.Alerted};
    END;

END Work.
