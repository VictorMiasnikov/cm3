INTERFACE FBEWr;

IMPORT FBE, Wr;

 Returns a fixed-width FBE.T that writes to the underlying Wr.T.  It does not
   understand tab characters; it treats them as being one position wide, just
   like other chars. 

PROCEDURE New (wr: Wr.T; width: INTEGER): FBE.T;

END FBEWr.
