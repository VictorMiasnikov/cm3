 The ErrMsg interface defines some useful procedures for generating
   meaningful error messages. 


INTERFACE ErrMsg;

IMPORT AtomList;

PROCEDURE Fatal(msg: TEXT; errList: AtomList.T := NIL);

 Emits an error message, and terminates the process. 

PROCEDURE GetErrno(errList: AtomList.T; VAR errno: INTEGER): BOOLEAN;

 If something resembling a Unix errno value is in the list, sets
   errno to its value and returns TRUE.  Otherwise, returns FALSE. 

PROCEDURE StrError(l: AtomList.T): TEXT;

 Returns a message corresponding to an error list from an OSError.E,
   Rd.Failure, or Wr.Failure exception. 

END ErrMsg.
