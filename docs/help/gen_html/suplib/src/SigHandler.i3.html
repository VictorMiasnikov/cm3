 The SigHandler interface allows Unix signals to be serviced in
   normal application code, where all the facilities of the thread
   scheduler and garbage collector are available. 


INTERFACE SigHandler;

TYPE
  T = OBJECT METHODS
    apply(sig: INTEGER);
  END;
  SigNum = [1..255];

PROCEDURE Register(sig: SigNum;
                   handler: T);

 Registers a handler for the signal sig.  If handler is NIL,
   restores the default action for sig. 

PROCEDURE Block();

 Suspends the handling of signals. 

PROCEDURE Unblock();

 Resumes handling signals. 

PROCEDURE ShutDown();

 Kills off the SigHandler.T. 

END SigHandler.
